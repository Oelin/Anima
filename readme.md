# Anima

A little scripting language that's easy to to read, write and learn. The frontend is nearly finished.


```ruby
def count(n)
  for i in range(n)
    print(i)
  end
end
```

### Features

* Weak typing
* Closures
* Lambdas
* IIFEs
* Pythonic for loops: `for <variable> in <iterator>`
* Minimal evaluation
